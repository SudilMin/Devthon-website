# Dev{thon} 3.0 Website

Welcome to the Dev{thon} 3.0 website! This project is dedicated to showcasing the details of the Dev{thon} 3.0 web competition, enabling team registrations, and providing information about previous editions of the competition.

## Overview

Dev{thon} is an exciting web competition that challenges participants to develop innovative solutions within a limited timeframe. The third edition, Dev{thon} 3.0, promises to be bigger and better, with more opportunities for collaboration and creativity.

### Previous Editions

- **Dev{thon} 1.0**: The inaugural event that set the stage for future competitions, focusing on creativity and teamwork.
- **Dev{thon} 2.0**: Building on the success of the first edition, this event introduced new challenges and attracted a larger pool of participants.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```
   git clone [repository-url]
   ```
2. Navigate to the project directory:
   ```
   cd devthon-3.0-website
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `src/index.html` in your browser to view the website.
